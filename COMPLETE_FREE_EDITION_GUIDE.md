## 🚀 すぐに実装できる高度なAI機能

### 1️⃣ Chain-of-Thought（CoT）
```
従来: "CVR が低い。LP を改善"
改善: "Step1: CVR低下検出→Step2: チャネル分析→Step3: 原因仮説→Step4: 改善案根拠付き"
効果: 精度 +30%, Hallucination -50%
工数: 1日
```

### 2️⃣ マルチエージェント
```
Analyst → Planner → Copywriter → Validator
→ Issues が見つかったら自動改善ループ
効果: Human review -60%, 品質 +40%
工数: 2-3日
```

### 3️⃣ Self-Consistency投票
```
T=0.5, 0.6, 0.7 で3回生成 → 多数決で最適案選択
効果: Hallucination -60%, 信頼度 +40%
工数: 1日
```

### 4️⃣ Vision LLM（LLaVA 7B）
```
スクリーンショット → テキスト+画像分析
効果: LP分析精度 +40%, UX提案が50%具体的に
工数: 30分（モデルDL）+ 1時間
```

### 5️⃣ RAG（記憶化）
```
過去分析結果を ChromaDB に保存
→ 同業界の問題に即座対応
効果: 処理速度 +50%, 品質一貫性 +30%
工数: 2時間
```

## 📖 使用開始手順

### 最もシンプルな開始方法（5分）

```bash
# 1. Ollama インストール（macOS/Linux）
curl https://ollama.ai/install.sh | sh

# 2. サービス起動 + モデルDL
ollama serve &
sleep 5
ollama pull llama3.1:8b

# 3. Marketing Auto Analyzer 起動
cd /home/nshigeoka/marketing-auto-analyzer
./start.sh
```

👉 これだけで **完全無料版が起動**（UI: http://localhost:8501）

### AI機能を段階的に有効化

```bash
# Basic (上記)のみ

# + Chain-of-Thought
echo "PROMPT_ENABLE_CHAIN_OF_THOUGHT=true" >> .env

# + マルチエージェント
echo "MULTI_AGENT_ENABLED=true" >> .env

# + Vision LLM
ollama pull llava:7b
echo "VISION_ANALYSIS_ENABLED=true" >> .env

# + Self-Consistency
echo "SELF_CONSISTENCY_ENABLED=true" >> .env

# 再起動
docker-compose restart
```
